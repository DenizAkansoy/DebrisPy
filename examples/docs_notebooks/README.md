# `docs_notebooks/` Directory

This repository includes a collection of Jupyter notebooks which are used when building the documentation. Note that these files can also be found inside the `docs/` directory, however we also include them here for easy access.

Modifying the notebooks in this folder **will not** affect the main documentation build, making it a safe environment for experimentation.

These notebooks are well-annotated, with clear, step-by-step examples showing how to initialise and use each of the core classes (e.g., `SigmaA`, `Kernel`, `ASD`, `MonteCarlo`). For clarity, we break down the `Eccentricity` class into 3 separate files based on the type of profile being used.
